# Tic‑Tac‑Toe 2‑Player Web

A browser‑based two‑player Tic‑Tac‑Toe game built with JavaScript, HTML and CSS.  
Perfect for quick head‑to‑head fun, this repo showcases a simple game logic implementation and UI responsiveness.

---

## Features
- Two‑player gameplay (player X vs player O)  
- Game board reset for a new round  
- Highlight of winning line when a player wins  
- Detection of draw when no more moves available  

---

## Tech Stack
- Vanilla JavaScript (ES6)  
- HTML5 & CSS3  
- No external backend or database (uses localStorage)  
- Hosted via static site deployment
  
--- 

## Notes
- Built as a **practice/demo project** to practice coding game logic in Javascript
- Uses Bootstrap for styling and responsiveness.  
- No further updates or maintenance planned; kept for reference.
